Match Pets
Match Pets é um aplicativo de encontro para cachorros baseado em Tinder, permitindo que os usuários possam encontrar novos amigos caninos ou até mesmo marcar encontros para cruzamentos.

Funcionalidades
Os usuários podem pesquisar cachorros por raça.
Os usuários podem dar like ou dislike em cachorros.
Os usuários podem visualizar os cachorros que deram like.
Os usuários podem visualizar os cachorros que deram dislike.
Os usuários podem visualizar os dados de um cachorro, como nome, idade, sexo e raça.
